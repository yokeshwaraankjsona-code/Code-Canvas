# CodeCanvas

CodeCanvas is a frontend-only interactive visual programming and algorithm visualization platform built with React, Vite, Tailwind CSS, Framer Motion, React Flow, and Lucide icons.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

This scaffold includes:

- Landing page
- Dashboard with a visual workspace (React Flow)
- Bubble Sort and Binary Search visualizers
- Tailwind CSS with glassmorphism and neon accents

Next steps: add more custom nodes, execution simulation, minimap, and export features.
